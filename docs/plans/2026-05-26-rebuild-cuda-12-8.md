# Rebuild CUDA 12.8 — onnxruntime-gpu con aceleración real

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Reconstruir el contenedor `onda:nvidia` para que `demucs-onnx` use GPU real (CUDA 12.8) en lugar de CPU fallback.

**Root cause:** `onnxruntime-gpu` 1.26.0 está instalado pero las librerías CUDA runtime del sistema (`libcudnn.so`, `libcublas.so`) no existen — PyTorch empaqueta las suyas privadas pero son invisibles para ONNX Runtime. `onnxruntime.get_available_providers()` devuelve `['AzureExecutionProvider', 'CPUExecutionProvider']` — sin `CUDAExecutionProvider` ni `TensorrtExecutionProvider`.

**Architecture:** Añadir repositorio NVIDIA CUDA 12.8 → instalar runtime libs (`cuda-cudnn-12-8`, `libcublas-12-8`, etc.) → set `LD_LIBRARY_PATH` → `onnxruntime-gpu` detecta CUDA. Mismo `Dockerfile` multi-stage existente, solo se modifica el stage `runtime`.

**Tech Stack:** python:3.12-slim (Debian Bookworm), CUDA 12.8 runtime libs from NVIDIA apt repo, onnxruntime-gpu 1.26.0, demucs-onnx 0.3.4

**Files:**
- Modify: `Dockerfile` (runtime stage, líneas 21-40)
- Read: `requirements-docker.txt`, `inference/inference_demucs_onnx.py`

### Task 1: Añadir repositorio NVIDIA CUDA 12.8 al runtime stage

**Objective:** Configurar `apt` para instalar paquetes CUDA 12.8 desde repositorio NVIDIA.

**Files:**
- Modify: `Dockerfile:21-28` (runtime stage)

**Step 1: Escribir el bloque apt para CUDA 12.8**

Reemplazar el bloque runtime actual (líneas 21-40) con la versión que añade NVIDIA repo + CUDA libs:

```dockerfile
# ── Runtime stage ────────────────────────────────────
FROM python:3.12-slim AS runtime

# NVIDIA CUDA 12.8 runtime libraries for onnxruntime-gpu
RUN apt-get update && apt-get install -y --no-install-recommends \
    wget gnupg \
    && wget https://developer.download.nvidia.com/compute/cuda/repos/debian12/x86_64/cuda-keyring_1.1-1_all.deb \
    && dpkg -i cuda-keyring_1.1-1_all.deb \
    && rm cuda-keyring_1.1-1_all.deb \
    && apt-get update \
    && apt-get install -y --no-install-recommends \
    cuda-cudnn-12-8 \
    libcublas-12-8 \
    libcublaslt-12-8 \
    libcufft-12-8 \
    libcurand-12-8 \
    libcusolver-12-8 \
    libcusparse-12-8 \
    && rm -rf /var/lib/apt/lists/*

# Runtime libraries path (onnxruntime-gpu busca aquí)
ENV LD_LIBRARY_PATH=/usr/local/cuda-12.8/lib64:${LD_LIBRARY_PATH}
ENV PATH=/usr/local/cuda-12.8/bin:${PATH}

RUN apt-get update && apt-get install -y --no-install-recommends \
    libsndfile1 \
    rubberband-cli \
    ffmpeg \
    libtk8.6 \
    && rm -rf /var/lib/apt/lists/*
```

**Step 2: Verificar que el Dockerfile es válido sintácticamente**


**Step 3: Commit**
