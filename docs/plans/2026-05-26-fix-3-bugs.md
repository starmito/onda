## Bug #1 — Presets sin nombre

### Causa
El struct `Preset` en `backend/internal/cli/flags.go` no tiene `json` tags. Go serializa con PascalCase (`"Name"`, `"Description"`) pero el frontend (`PresetSelector.svelte`) lee `preset.name` y `preset.description` (camelCase).

### Fix
Añadir `json:"name"` y `json:"description"` al struct `Preset` en `flags.go`.

### Verificación
`curl http://192.168.1.87:3000/api/models` debe devolver claves `"name"` y `"description"` (minúsculas).

## Bug #3 — GPU no detectada

### Causa
El contenedor `onda` en .87 se recreó con `docker compose up -d` (sin el override nvidia). Resultado: runtime `runc`, sin devices GPU.

### Fix
Recrear contenedor en .87 con:
```
docker compose -f docker-compose.yml -f docker-compose.nvidia.yml up -d onda
```

### Verificación
`curl http://192.168.1.87:3000/api/health` debe devolver `"gpu":true`.
